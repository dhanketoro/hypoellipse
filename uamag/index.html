U of Alaska Calibration Routines

The program prm2bin_cal converts an ASCII calibration file into a binary file for use with the UNIX version of Hypoellipse.  The scratch file named fort.14 may be deleted after the run.

The program bin2prm_cal will convert a binary file back into ASCII format.
